# 🕊️ Blue Wing — Sovereign Agentic Entity

Blue Wing is a high-performance, interactive AI agent architecture designed to operate as an Industrial OS. It integrates 3D holographic visualization, low-latency voice interaction, and autonomous Web3 execution.

## 🌌 System Architecture

- **Perception Layer**: Web Speech API + Vision Feed (Camera optics).
- **Cognition Layer**: Next.js API (Claude 3.5 Sonnet / Gemini Flash integration).
- **Action Layer**: Hardhat (Web3) + Appwrite (Cloud Memory).
- **Body**: Three.js GPGPU Particle Core with state-driven animations.

## 🛠️ Operational Protocols

### Keyboard Shortcuts
- **Alt + B**: Wake / Focus Command Bar
- **Alt + S**: Split (Multi-Agent Cluster)
- **Alt + D**: Deep Sleep (Low Power)
- **Alt + M**: Toggle Mic (Voice Control)

### Authority Overrides
- **"God-Mode"**: Toggles the system-wide visual transformation (Gold/White). Grants authority override visual protocols.

## 📦 Tech Stack

- **Frontend**: Next.js 15 (App Router)
- **3D Engine**: React Three Fiber + Drei + Postprocessing
- **Voice**: Web Speech API (Recognition + Synthesis)
- **Database**: Appwrite (Cloud Vault)
- **Blockchain**: Hardhat + Ethers.js
- **Styling**: CSS Modules + Glassmorphism

## 🚀 Getting Started

1. **Environment Setup**:
   Add your Appwrite keys to `.env.local`:
   ```env
   NEXT_PUBLIC_APPWRITE_ENDPOINT="https://cloud.appwrite.io/v1"
   NEXT_PUBLIC_APPWRITE_PROJECT_ID="YOUR_PROJECT_ID"
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access HUD**:
   Open [http://localhost:3000](http://localhost:3000)
